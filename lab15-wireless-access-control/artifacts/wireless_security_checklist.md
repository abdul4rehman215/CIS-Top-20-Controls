# Lab 15 - Wireless Access Control (Checklist)

Date: 2026-02-28

## Task 1 - Access Router Admin Page
- Router IP identified (Default Gateway): 192.168.1.1
- Admin login successful via browser

## Task 2 - Enable WPA2/WPA3 Encryption
- Security mode set to: WPA2-PSK (AES)
- Strong passphrase set (12+ chars with complexity)
- Settings saved/applied and Wi-Fi restarted

## Task 3 - Disable WPS
- WPS located under Wireless/Advanced settings
- WPS set to Disabled
- Settings saved/applied

## Task 4 - Verify Unauthorized Access Prevention
- Unauthorized device tested with incorrect password → authentication failed
- Authorized device reconnected successfully using new passphrase
